JSON User Database Authentication Example.


JSON User Database Authentication.

POST to /ps1/auth/userdb.ps1 JSON structured as

{"Username":"SENT_USERNAME","Password":"SENT_PASSWORD"}

to check userdb.json structured as

[ { "Username": "USERNAME1", "Password" : "PASSWORD" },
{ "Username": "USERNAME2", "Password" : "PASSWORD" } ]

Correct password will return {"Credentials": "Valid"}
Incorrect password will return {"Credentials": "Invalid"}
